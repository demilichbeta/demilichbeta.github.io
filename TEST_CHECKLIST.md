## 03:00異常檢查

- [ ] CS任一站統計差異不為0時，按複製會跳警告。
- [ ] SS或KS任一站統計差異不為0時，按複製會跳警告。
- [ ] 只有NS、TS或E系列異常時，不會影響03:00警告判定。

## 05:00異常檢查

- [ ] NS任一站統計差異不為0時，按複製會跳警告。
- [ ] TS任一站統計差異不為0時，按複製會跳警告。
- [ ] ET、HT或YT任一站統計差異不為0時，按複製會跳警告。
- [ ] 只有CS、SS或KS異常時，不會影響05:00警告判定。

## 警告介面

- [ ] 警告只顯示存在數字差異異常，不列出站所名稱。
- [ ] 有異常時不會立即複製。
- [ ] 按「仍要複製回報文字」後可以複製。
- [ ] 按右上角×可關閉。
- [ ] 切換03:00／05:00後警告消失。
- [ ] 移到夜班、盤點、統計等其他功能後警告消失。

## 離線與資料

- [ ] 有網路開啟一次後，斷網仍能啟動。
- [ ] 關閉 App 再開啟，資料仍保留。
- [ ] CSV、JSON與工作日誌匯出正常。
