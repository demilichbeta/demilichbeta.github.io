# Android實機測試清單｜v4

## A. 更新與資料

- [ ] 左上選單底部顯示「版本 v4.0」。
- [ ] v3原有站所數量仍存在。
- [ ] v3回倉批次已換算成各來源回倉總數。
- [ ] 關閉App再開啟，資料仍保留。

## B. 夜班與復原

- [ ] 夜班頁按站所一次，數量＋1。
- [ ] 復原提示只在夜班頁出現。
- [ ] 按右側✕可關閉復原提示。
- [ ] 切換到載走、盤點或統計後，復原提示消失。
- [ ] 復原剛才的夜班操作後，數量正確減回。

## C. 修正前面板數

- [ ] 點「修正前面板數」後，顯示每站大型−1／＋1。
- [ ] 可直接修正2～3步以前的站所，不影響後面其他站所。
- [ ] 數量為0時按−1不會變成負數。
- [ ] 點「返回快速＋1」可回到一般夜班頁。
- [ ] 離開夜班頁再回來，預設回到快速＋1模式。

## D. 回倉

- [ ] 顯示DC9、DC4、DC11、CS12、SDC、DC2、NS2、NS1、DC12。
- [ ] 每個來源按＋1可增加板數。
- [ ] 每個來源按−1可修正板數。
- [ ] 數量為0時不能再減。
- [ ] 頁面只顯示來源數量與總板數，不顯示時間或過境總數。

## E. 過境與底部導覽

- [ ] 底部只有夜班、載走、盤點、統計四項。
- [ ] 左上其他功能中可找到「過境貨」。
- [ ] 過境仍可用−1／＋1調整。
- [ ] 過境數量仍正確計入統計總數。

## F. 快速報表與滑道

- [ ] 快速回報頁只顯示03:00 CS／SS／KS。
- [ ] 沒有05:00切換按鈕。
- [ ] 統計頁仍可查看NS／TS及全部總數。
- [ ] 滑道名稱依序為第七、第八、第九、第十、第十一、第十二。

## G. 離線與匯出

- [ ] 開啟一次後關閉網路，App仍能啟動。
- [ ] 點貨事件CSV可下載。
- [ ] 回倉CSV只包含來源與回倉板數。
- [ ] JSON備份可匯出並重新匯入。
- [ ] 工作日誌中的回倉紀錄不含時間或過境批次。
