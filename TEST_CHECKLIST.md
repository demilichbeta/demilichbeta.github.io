## 中班複製文字

- [ ] 只有籠車時顯示「2籠」，不是「籠2」。
- [ ] 只有棧板時顯示「3板」，不是「板3」。
- [ ] 籠板都有時顯示「1籠／1板」。
- [ ] 合計顯示「X籠／X板｜總X」。

## 快速回報

- [ ] 03:00畫面只顯示中、夜、過、總，沒有二分。
- [ ] 05:00畫面只顯示中、夜、過、總，沒有二分。
- [ ] 複製文字內沒有「二」或「二分」。
- [ ] 二次分理已有數量時，03:00／05:00總數仍不包含二分。

## 異常提示

- [ ] 尚未盤點時按複製，不會把全部站所誤判異常。
- [ ] 盤點籠車數量不符時，提示框列出該站及「籠應X／現X」。
- [ ] 盤點棧板數量不符時，提示框列出該站及「板應X／現X」。
- [ ] 明確輸入0但應有數不為0時，仍會提示異常。
- [ ] 有異常時不會立即複製，按「仍要複製回報文字」後才複製。
- [ ] 按提示框右上角×可關閉。
- [ ] 切換03:00／05:00後提示框消失。
- [ ] 移到夜班、盤點、統計等其他功能後提示框消失。

## 離線與資料

- [ ] 有網路開啟一次後，斷網仍能啟動。
- [ ] 關閉 App 再開啟，資料仍保留。
- [ ] CSV、JSON與工作日誌匯出正常。
