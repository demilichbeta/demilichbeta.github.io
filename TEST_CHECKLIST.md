# Android 實機測試清單（v2）

## A. 安裝與更新

- [ ] Android Chrome 可正常開啟 HTTPS 網址。
- [ ] 可按「安裝」加入手機主畫面。
- [ ] 直式深色介面沒有左右超出。
- [ ] 從 v1 更新後，原有班次資料仍存在。
- [ ] 底部五個功能鍵未被 Android 導覽列遮住。

## B. 夜班與站所

- [ ] 25 個站所及五個滑道順序正確。
- [ ] 按 NS2 一次，夜班由 0 變 1。
- [ ] 有短震動及成功效果。
- [ ] 復原上一筆後，NS2 回到 0。

## C. 過境 −1／+1

- [ ] 點選站所後只顯示大型 −1 與 +1。
- [ ] 按 +1 後過境數量增加一。
- [ ] 按 −1 後過境數量減少一。
- [ ] 過境為 0 時按 −1，不會變成負數。
- [ ] 過境 −1 與 +1 都會寫入事件紀錄。
- [ ] 可復原上一筆過境調整。

## D. 線上整批功能

- [ ] 「全部 25 站＋1」後，每站線上數量都增加一。
- [ ] 復原上一筆後，25 站全部恢復。
- [ ] 「全部線上轉完成」後，全部線上歸零並加入夜班。
- [ ] 復原上一筆後，全部線上及夜班數量一起恢復。
- [ ] 單站 −1、+1及轉完成仍正常。

## E. 早班大型控制

- [ ] 每站都有大型 −／＋按鈕。
- [ ] 按鈕在手機上容易單手點擊。
- [ ] 按 + 後早班增加一。
- [ ] 按 − 後早班減少一。
- [ ] 早班為 0 時不會再減成負數。
- [ ] 點中間數字可開啟 Android 數字鍵盤直接輸入。
- [ ] 離開輸入欄位後自動保存。

## F. 統計公式

以某站輸入：早班5、夜班4、過境2、線上1、載走3、現場8。

- [ ] 回報總數為12。
- [ ] 應在現場為9。
- [ ] 差異為−1。
- [ ] 差異不為零時醒目顯示。
- [ ] 現場改為9後顯示正常。

## G. 持久化與離線

- [ ] 關閉 App 再開啟，資料仍存在。
- [ ] 飛航模式下可從主畫面開啟。
- [ ] 離線時可新增及修改紀錄。
- [ ] CSV 可下載。
- [ ] JSON 可備份及還原。
- [ ] 工作日誌文字可複製。
