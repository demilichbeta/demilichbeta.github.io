# 物流夜班點貨 PWA v8

Android Chrome 可安裝的離線自用點貨工具。

## v8 重點

1. 過境與回倉的載具切換列會固定在畫面上方，往下滑仍可切換。
2. 二次分理的待處理數不進入統計；按下「轉完成」後才會進入「二分」及回報總數。
3. 二次分理頁底部顯示各站已轉完成數量。
4. 底部新增「回報」，可在頁首切換03:00與05:00。

## 更新方式

更新前先匯出完整 JSON，將 v8 全部檔案覆蓋原網站，完全關閉 App 後重新開啟。看到左側選單「版本 v8.0」代表更新成功。

## 本機測試

在此資料夾開啟命令提示字元：

```bat
py -m http.server 8080
```

Chrome 開啟 `http://localhost:8080`。
