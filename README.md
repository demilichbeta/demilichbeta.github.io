# 物流夜班點貨 PWA v6

Android Chrome 可安裝、可離線使用的自用點貨工具。資料保存在該瀏覽器的 IndexedDB，不需要登入或雲端資料庫。

## v6 主要更新

1. 回倉來源新增 **NS5**。
2. 回倉頁新增「特殊狀況備註」：文字會自動歸入儲存當下的 30 分鐘時段，可在時段卡片中查看及刪除，也會輸出至回倉 CSV 與工作日誌。
3. 04:30「全部＋1」只套用 **NS／TS 共 25 站**，不包含 CS、SS、KS、YT、HT、ET。
4. 現場盤點改為字母系列排序，可切換全部、CS、ET、HT、KS、NS、SS、TS、YT。
5. 統計頁在「全部站所」時依字母系列分區顯示，不再按照滑道排列。
6. 所有畫面、回報、CSV及工作日誌中的「早班／早」改為「中班／中」。
7. 線上頁仍依第七至第十二滑道排列，並以灰色滑道標題分隔。

## 從 v5 更新

1. 先在 v5 的「班次／備份」匯出完整 JSON。
2. 將 v6 全部檔案覆蓋原網站空間。
3. 完全關閉已安裝的 App，再重新開啟。
4. 左上選單底部看到「版本 v6.0」即更新成功。
5. 若仍顯示舊版，關閉 App 後再開一次；不要清除 Chrome 網站資料。

v6 沿用原本資料庫。v5 的 `morning` 內部欄位會保留，只是使用者介面統一顯示為「中班」，不會遺失舊數字。

## 電腦本機測試

在包含 `index.html` 的資料夾開啟命令提示字元：

```bat
python -m http.server 8080
```

若 `python` 無法執行：

```bat
py -m http.server 8080
```

Chrome 開啟：

```text
http://localhost:8080
```

## Android 安裝

PWA 必須透過 HTTPS 網址部署。用 Android Chrome 開啟網址後：

1. 點右上角選單。
2. 選「安裝應用程式」或「加到主畫面」。
3. 首次上線開啟完成後，Service Worker 會快取 App 檔案供離線使用。

## 資料安全

- 每次操作會自動保存。
- 更新前先匯出完整 JSON。
- 清除 Chrome 網站資料會刪除本機工作資料。
- 建議定期匯出 JSON 與 CSV。
- 電拖必須完全停止後再操作手機。
